# TourMate – Tour Guide Booking App

TourMate is a mobile application developed using Flutter (Dart) and Firebase in Android Studio.  
The app helps travelers easily find and book local tour guides for their trips.

## Features

- User Registration & Login
- Browse Available Tour Guides
- View Guide Details
- Book Tour Guides
- Firebase Authentication
- Realtime Database / Firestore Integration
- Clean and User-Friendly UI
- Mobile Responsive Design

## Technologies Used

- Flutter
- Dart
- Firebase
- Android Studio

## Firebase Services

- Firebase Authentication
- Cloud Firestore / Realtime Database
- Firebase Storage

## Installation

1. Clone the repository

git clone https://github.com/Salwa101/TourMate.git


2. Open the project in Android Studio

3. Install dependencies

   flutter pub get

4. Run the app


flutter run
